Introduction

This webservice tool was built on the idea that we as consumers need more guidance in the expertise of stocks and knowing when to buy/sell them. For the puposes of this tool, we used imported stock prices in the past from AlphaVantage. With these prices, we used machine learning to predict future stock prices over a given time window. The implementation was done in primarily HTML, CSS, JavaScript with Python for machine learning and the Bootstrap CSS framework.

Frequently Asked Questions and Issues

  1. Why can't I connect to the Web Application?
    • Since the tool uses the Django server, you must be connected to it first. Open up the directy where the source code for this tool is in the command line and enter the following command:
      • python manage.py runserver
    • You might have the wrong username and password
    • You might not be connected to the internet
    • You must be using Google Chrome, Safari, Firefox, or Internet Explorer. Any other browser will not support it

  2. Why is it loading so slowly if at all?
    • If all the pages are loading slowly then it might be a connection issue. Restart server and computer, then try again
    • If search page is loading slowly then it is because the API used pulls 5 stocks every minute. Takes full minute till search can be redone.
    • If the saved stock page is loading slowly, it is also an API issue where it needs a full minute or the API might be down

  3. Why does Dashboard not alert me of my stock changes?
    • The API might be down
    • You need to save stocks first, since this page pulls changes from your saved stocks page.

  4. Why does the Search Page only load 5 stocks?
    • Unfortunately, that is just the API used. AlphaVantage only allows users to pull 5 stocks every 60 seconds (1 minute).
  5. Why is the Search Page loading so slowly?

As a background to using the tool, you must have Django installed, as well as Python. The user login information is stored on Django's builtin user data table. Once you have Django and Python installed, the user can run the Webservice by entering the following command into the command line FROM THE DIRECTORY WHERE THE SOURCE PROJECT FILES ARE SAVED: 'python manage.py runserver'

After connecting to the server, the user can enter the URL given in the command prompt window and use their credentials to log in. If first time user, then use admin as user and password as password. This will open up the admin interface where the user can add themselves as a new user.

Key features:

  • Predicts future stock prices based on user input of time window over which prediction is made, using machine learning.
  • User can save stocks directly from the "Search Stocks" page. Once saved, they will show up in the "Saved Stocks" page.
  • User can search stocks in the "Search Stocks" page by entering any keyword for the stock. They can even add filter by or order by criteria to sort the stocks in a certain fashion.
  • Upon each login, dashboard (home) page will load and alert user of what they missed since last log-in. This alert is in the form of price increase/decrease from ONLY their saved stocks.
  • If user wants to know what a particular stock is, then they can simply hit the "Details" button in the individual stock container.
  • User can also see the past trends of the stock in the form of a line graph. This populates from live data and is in the icon window of the stock container.
