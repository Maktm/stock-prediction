Introduction

This webservice tool was built on the idea that we as consumers need more guidance in the expertise of stocks and knowing when to buy/sell them. As a background to using the tool, you must have Django installed, as well as Python. The user login information is stored on Django's builtin user data table. Once you have Django and Python installed, the user can run the Webservice by entering the following command into the command line FROM THE DIRECTORY WHERE THE SOURCE PROJECT FILES ARE SAVED: 'python manage.py runserver'

After connecting to the server, the user can enter the URL given in the command prompt window and use their credentials to log in. If first time user, then use admin as user and password as password. This will open up the admin interface where the user can add themselves as a new user.

Key features:

  • Predicts future stock prices based on user input of time window over which prediction is made, using machine learning.
  • User can save stocks directly from the "Search Stocks" page. Once saved, they will show up in the "Saved Stocks" page.
  • User can search stocks in the "Search Stocks" page by entering any keyword for the stock. They can even add filter by or order by criteria to sort the stocks in a certain fashion.
  • Upon each login, dashboard (home) page will load and alert user of what they missed since last log-in. This alert is in the form of price increase/decrease from ONLY their saved stocks.
  • If user wants to know what a particular stock is, then they can simply hit the "Details" button in the individual stock container.
  • User can also see the past trends of the stock in the form of a line graph. This populates from live data and is in the icon window of the stock container.
